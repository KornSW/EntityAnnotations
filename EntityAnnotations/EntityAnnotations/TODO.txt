

EDMX transpiler   T4, das aus einer edmx in eine modeldefinitin(xml) datei generitt

EDMX Entity generator (kapselt EDMX transpiler + Entity GENERATOR)

Entity GENERATOR T4, welches aus einer modeldefinitin(xml) entitied muit attributen generiet

modeldefinition extractor  T4, das aus einem kompilat (menge von enetity-typen) modeldefinitin(xml) datei generitt

EF-Codefirs mapping CONFIGURATOR  (zur laufzeit model lesen und ef konfigurere)
     variente 1: unterstütztzt as quelle ein modeldefinitin(xml)
     variente 2: unterstütztzt as quelle ein EDMX (kapselt EDMX transpiler + variente 1)
     variente 2: unterstütztzt as quelle ein kompilat (menge von enetity-typen) (kapselt modeldefinition extractor  + variente 1)

EF-Codefirs mapping code GENERATOR  (ef mappingcodde generieren)
     variente 1: unterstütztzt as quelle ein modeldefinitin(xml)
     variente 2: unterstütztzt as quelle ein EDMX (kapselt EDMX transpiler + variente 1)
     variente 2: unterstütztzt as quelle ein kompilat (menge von enetity-typen) (kapselt modeldefinition extractor  + variente 1)
